= Feature matrix and licensing options
:toc: true


:page-title: Feature matrix and licensing options
:page-pageid: license-feature-matrix
:page-description: Choose the license model that suits your embedding requirement.

ThoughtSpot offers a separate license for customers who want to embed ThoughtSpot functionality in their application, web page, or portal. The ThoughtSpot Everywhere Edition license allows embedding live search, dynamic visualizations and Liveboards, and the entire ThoughtSpot experience within an app. This license also provides full access to the interactive Developer portal, APIs and SDK, and customization and security controls.

[#purchase-license]
== Get ThoughtSpot Everywhere Edition license

If you already have a ThoughtSpot Enterprise Cloud service subscription::

Users with Developer or admin privileges can access the *Develop* tab in the ThoughtSpot UI. However, if you do not have the ThoughtSpot Everywhere license, the **Customization** features such as *Security settings*, and *Link settings* are locked. To unlock these features, you must purchase the ThoughtSpot Everywhere Edition license. 

+
With ThoughtSpot Everywhere Edition license, you can get full access to the Developer portal, Playground, APIs and SDK, customization controls, and security settings. 


If you do not have a ThoughtSpot Enterprise Cloud Service subscription::

We recommend that you sign up for a free trial, try out ThoughtSpot features, and then upgrade your license. To avail ThoughtSpot Everywhere features, you will require the ThoughtSpot Everywhere Edition license along with a ThoughtSpot Enterprise Cloud service subscription. 

For more information, see xref:get-started-tse.adoc[Get started with ThoughtSpot Everywhere].
