By default, the SDK fetches resources when they are required. If you want to prefetch and cache static resources, and serve these resources from the user's local disk, you can enable the prefetch API call.

The prefetch API adds a frame to prefetch the resources before the application loads. The prefetched resources are then cached locally to serve the application pages faster.  

[source,javascript]

----
 prefetch(url): "https://<hostname>:<port>",
 callPrefetch: true,
----
prefetch(url)::
_String_. Hostname or IP address of the URL from which you want prefetch resources.

callPrefetch::
_Boolean_. When set to `true`, the SDK calls the prefetch API to prefetch the resources from the specified URL.
