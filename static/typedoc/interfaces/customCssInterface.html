Optional rules_UNSTABLE

rules_UNSTABLE: {}
  • Defined in types.ts:221

Can be used to define a custom font face like:

example
rules_UNSTABLE?: {
    "@font-face": {
        "font-family": "custom-font",
        "src": url("/path/")
    };
  };

Also, custom css rules outside of variables.

example
rules_UNSTABLE?: {
    ".thoughtspot_class_name": {
        "border-radius": "10px",
        margin: "20px"
    };
  };
