    Navigate to particular page for app embed. eg:answers/pinboards/home This is used for embedding answers, pinboards, visualizations and full application only.

    version

    SDK: 1.12.0 | ThoughtSpot: 8.4.0.cl, 8.4.1-sw

    Parameters

    • path: string | number

      string | number The string, set to iframe src and navigate to new page eg: appEmbed.navigateToPage('pinboards') When used with noReload this can also be a number like 1/-1 to go forward/back.

    • noReload: boolean = false

      boolean Trigger the navigation without reloading the page
